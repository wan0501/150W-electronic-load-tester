## 项目概述

电子负载仪是一种模拟真实负载的测试仪器，广泛用于电源、电池、充电器等设备的性能测试。本设计以 STM32G431CBT6 为核心，利用其内置 **12 位 DAC** 和 **12 位 ADC** 实现精确的恒流 / 恒压控制。

### 主要特点

- **双工作模式**：恒流（CC）和恒压（CV）模式
- **全数字控制**：STM32G431 DAC 输出设定值，ADC 采样反馈，实现闭环调节
- **TFT 彩屏显示**：实时显示电压、电流、功率、温度等参数
- **按键交互**：3 个独立按键（SW1、SW2、SW3）进行模式切换和参数设置
- **过温保护**：NTC 热敏电阻实时监测功率管温度
- **串口通信**：USART1 接口，支持上位机控制

## 系统架构

### 工作原理

```
                 ┌─────────── 功率MOSFET ───────────┐
                 │   (线性工作区，耗散功率)            │
                 │                                   │
 被测电源 ────►┤ 输入保护  ├───►┤ 功率管 ├───►┤ 地  │
                                           │
                              ┌────────────┤
                              │            │
                         ┌────▼────┐  ┌───▼────┐
                         │ 10mΩ   │  │ NTC    │
                         │ 采样电阻│  │ 测温   │
                         └────┬────┘  └───┬────┘
                              │            │
                     ┌────────▼────────────▼───────┐
                     │   运放信号调理              │
                     │   CC: 40x 放大              │
                     │   CV: 11x 放大              │
                     └────────┬───────────────────┘
                              │
                     ┌────────▼───────────────────┐
                     │   STM32G431CBT6            │
                     │   ┌─────────────────────┐  │
                     │   │ DAC_OUT1 (PA4) → CV  │  │
                     │   │ DAC_OUT2 (PA5) → CC  │  │
                     │   │ ADC1_IN1 (PA1) → CC  │  │
                     │   │ ADC1_IN2 (PA2) → CV  │  │
                     │   │ ADC1_IN6 (PA6) → NTC │  │
                     │   │ USART1 (PA9/PA10)    │  │
                     │   │ TFT SPI 显示         │  │
                     │   │ 按键 SW1~SW3         │  │
                     │   └─────────────────────┘  │
                     └────────────────────────────┘
```

### 控制原理

- **恒流（CC）模式**：STM32G431 DAC_OUT2 输出设定电压 → 40 倍运放 → 与 1.24V 基准比较 → 驱动功率管栅极 → 使电流等于设定值
- **恒压（CV）模式**：STM32G431 DAC_OUT1 输出设定电压 → 11 倍运放 → 与被测电压比较 → 驱动功率管栅极 → 使电压等于设定值

### 恒流（CC）环路

CC 环路控制流过功率管的电流：

- **DAC 设定**：DAC_OUT2 (PA5) 输出 0~3.3V（12 位分辨率）
- **电流检测**：10 mΩ 精密采样电阻
- **信号放大**：40 倍运放放大 10 mΩ 上的压降
- **基准电压**：1.24 V 基准
- **输出能力**：DAC 值 1500 ~ 4095 对应电流 ~0.11 A ~ 5.16 A

**CC 模式电流计算**：

```
I_SET = (DAC_OUT2 × 40) / (10mΩ × 40)  ≈ DAC_OUT2 / 10mΩ
```

当 DAC 输出 2.02V 时，40 倍放大后为 80.8V 基准 → 经过电流环路稳定在约 1.93A。

### 恒压（CV）环路

CV 环路控制被测电源两端的电压：

- **DAC 设定**：DAC_OUT1 (PA4) 输出 0~3.3V（12 位分辨率）
- **反馈放大**：11 倍运放
- **输出能力**：DAC 值 50 ~ 4095 对应电压 ~0.44 V ~ 30 V

**CV 模式电压计算**：

```
V_SET = DAC_OUT1 × 11
```

### 显示与交互

- **TFT 彩屏**：SPI 接口（CS / SCLK / SDA / DC / RST / BL）
- **物理按键**：SW1、SW2、SW3（PC13~PC15，内部上拉）
- **LED 指示灯**：PA15，低电平点亮，用于状态指示

### 保护与采样

- **过温保护**：NTC 热敏电阻由 PA6 ADC 采样，实时监测功率管温度
- **过流保护**：硬件电流检测 + MCU ADC 监控
- **风扇控制**：预留控制接口（根据温控策略）

## 使用说明

### 查看设计

1. **原理图**：使用立创EDA 打开 `.eprj2` 工程文件，或直接查看 `负载仪原理图.pdf`
2. **仿真**：使用 NI Multisim 14 打开 `.ms14` 文件

### 固件开发

- 使用 **Keil MDK** 或 **STM32CubeIDE** 开发
- MCU：STM32G431CBT6（ARM Cortex-M4, 170MHz）
- 外设驱动：DAC、ADC、USART、SPI、GPIO

### 串口驱动

- 如果使用串口通信，运行 `CH341SER (1).EXE` 安装 CH340/CH341 驱动

### 注意事项

- **散热**：150W 满功率运行时功率管发热量大，必须加装足够的散热片和风扇
- **功率管工作在线性区**，不是开关模式，功耗全部转化为热量
- 输入电压和电流请勿超过额定值（30V / 5A / 150W）
- 首次使用建议从小功率开始测试
